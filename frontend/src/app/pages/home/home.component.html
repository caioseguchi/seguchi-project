Project Overview

During my Higher Diploma in Science in Computing at NCI, I gained a solid foundation in software development. However, I realized that I still had some knowledge gaps—especially in Spring Boot and Angular—that I wanted to close. To grow as a developer, I decided to learn by doing. This project was born from that decision: a fullstack web application built from scratch to sharpen my backend and frontend skills while exploring real-world concepts.

Tech Stack & Features

This website is a fullstack application built with Angular (frontend) and Spring Boot (backend), and deployed on AWS. Key features include:

  • User authentication (sign up & login) Role-based access control (Admin & User)
  • Task management system with full CRUD (create, read, update, delete) operations
  • Personalized dashboard: users manage their own tasks (with day, hour, description)
  • Admin panel: view all users, search accounts, and delete accounts when needed
  • External API consumption
