Project Overview

This site is a fullstack web application built with Angular and Spring Boot. It features authentication, role-based access (Admin & User), and project management functionality.

Authenticated users can register, log in, and manage their personal task list — similar to a calendar — with CRUD operations for tasks (day, hour, description). Users can only access and edit their own tasks.

Administrators have extended privileges: they can view all users, perform user searches, and delete accounts when necessary.
